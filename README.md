## 📌 Project Overview

Internly is an Internship Tracking System that helps monitor the complete internship process—from student applications to interview rounds and final offers.

This project demonstrates an end-to-end Data Analytics workflow including:

- Data Collection
- SQL Database Design
- Data Analysis
- Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboard
- Business Insights

---

## 🛠 Tech Stack

- **SQL (MySQL)** – Database Design & Analysis
- **Python** – Pandas, NumPy, Matplotlib
- **Jupyter Notebook** – EDA
- **Power BI** – Dashboard & Visualization
- **CSV Dataset**
